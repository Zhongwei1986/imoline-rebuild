# imOnline
-----------------------
## 如何使用
```
$ cd imOnline
$ npm install
$ node index
```
用浏览器访问:  `http://localhost:3000`

## 技术栈
+ 客户端：
  + HTML
  + CSS
  + JavaScript
+ 服务端
  + 全局环境:Node.js
  + Web服务器框架:express
  + 双向通信框架:socket.io
  + 数据库: MongoDB
